API and Event handling

Time Related Functions

GetHour
Prototype
int GetHour();
PurposeReturns the hour of the day
Example of Usage
var iHour = GetHour();

 

GetMinute
Prototype
int GetMinute();
PurposeReturns the minute of the day
Example of Usage
var iMinute = GetMinute();

 

GetDay
Prototype
int GetDay();
PurposeReturns the day number of the server (UO days since start)
Example of Usage
var iDay = GetDay();

 

GetSecondsPerUOMinute
Prototype
int GetSecondsPerUOMinute();
PurposeReturns the number of real seconds which equal a UO minute
Example of Usage
var iNumSecs = GetSecondsPerUOMinute();

 

GetCurrentClock
Prototype
int GetCurrentClock();
PurposeReturns server's uiCurrentTime, or number of clock ticks elapsed since startup
Example of Usage
var iTime = GetCurrentClock();
