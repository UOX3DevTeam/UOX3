SpawnNPC
Prototype
CHARACTER SpawnNPC( bpSocket, region, npcNum, worldNumber );
CHARACTER SpawnNPC( Socket, nnpcNum, x, y, z, worldNumber );
PurposeCreates a character based on a script entry (npcNum / nnpcNum)
Parameter explanationbpSocket is the socket of the character to add with (although this can be -1). If region is not -1, then this is the spawn region that is spawned from. npcNum is the entry # in npc.scp to spawn from.
Socket is the socket where it is being spawned from (can be -1). It is only not -1 when you have just used a target (basically pop up targeting of location). nnpcNum is the entry # in npc.scp to spawn. x, y and z is the target location (if socket != -1) to spawn at. worldNumber is the world that it will spawn in (0 == britannia, 2 == ilshenar)
Example of Usage
var iMade = SpawnNPC( -1, -1, 1000, 0 );	// makes a guard
var iMade2 = SpawnNPC( -1, 1000, 5900, 2000, 0 );	// makes a guard in green acres

{

	TextMessage( pTalkingTo, "You're a bonza bloke!" );

}

 

SetGender
Prototype
SetGender( pCheck, iValue );
PurposeiValue can either be 0 (male, dead or alive) or 1 (female, dead or alive). Any other values results in no action taken.
Example of Usage
if( GetGender( pTalking ) == 0 )

{

	SetGender( pTalking, 1 );	// Sex change from man to woman! :)

}
