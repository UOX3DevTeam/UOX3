CreateBuffer
Prototype

PurposeCreates a buffer of length len
Example of Usage

DestroyBuffer
Prototype
DestroyBuffer( buffer );
PurposeDestroys memory allocated with CreateBuffer
Example of Usage

SendBuffer
Prototype

PurposeSends buffer bufferID to socket
Example of Usage
