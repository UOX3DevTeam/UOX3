Print
Prototype
void Print( string );
Purpose Note: Include \n at the end of the string unless you're sending multiple messages
Example of Usage
console.Print( "Hello World!\n" );
Log
Prototype
void Log( string );
void Log( string, filename );
Purpose Note: Include \n at the end of the string unless you're sending multiple messages
Optional second parameter is the name of the file to log the message into
Example of Usage
console.Log( "I found an error, and I'm logging it!" );

Reload
Prototype
void Reload( type )
Purpose

Reload something from console. Available types:
0 Reload INI
1 Reload Accounts
2 Reload Regions
3 Reload Spawn Regions
4 Reload Spells
5 Reload Commands
6 Reload Definition Files
7 Reload JS
8 Reload HTML

Example of Usage
console.Reload( 4 ); //Reloads spells
