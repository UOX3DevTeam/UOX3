How do I create a playable orc race?

Orc race, daemon race, cat race, dog race are all possible. But any graphic other than human will make your paperdoll blank.

You will have to guess where swords are, or armor or cloths and they also won't show on your character in game.

So beware...

First make a gate or some item to change the player

'ADDITEM 2523

Now set it to a monster gate

'SETTYPE 82

Now set it to the type of monster you want

'setmorex # //#= the npc number in npc.scp

So an orc would be

'setmorex 69
