How can I make dragons rideable?

You cannot.

The only solution that has been really realized so far is replacing on each user-client a horse graphics with modified dragon sprites.

All other talking like server-side entrainment of a second creature with the player are only theoretic, and I doubt them to be ever possible. And even if it could function every one agrees that it would be very ineffective and lag prone.

Also from what I know of the client I believe it would look terrible wrong in all action animations.

Rumors are around that OSI themselves already wanted to make them rideable and failed. Maybe with Third Dawn they've reserved themselfes (and so also for us ) a possibily for hauler-dragons.
