1.9 - SERVER BACKUPS
    UOX comes with an ability to set an archive directory for the wsc files. Just          before the world
    is saved, the old wsc files are placed into the directory specified by a parameter in
    the uox.ini file. To make this entry in uox.ini, start UOX3 and then shut down the server again.
    Now look in uox.ini. There should be an entry called BACKUPDIRECTORY. If BACKUPDIRECTORY is blank then no
    archiving will take place. To specify a location to archive the wsc file at, enter a path such as this:
    BACKUPDIRECTORY=C:\SHARDBACKUP\ (the ending backslash IS required.)
    The file name will be in the format: uox3-yyyydddhhmmss.wsc where yyyy=4 digit year, ddd=3 digit day of year, etc.
    The Default setting for server backups is NO archiving.
