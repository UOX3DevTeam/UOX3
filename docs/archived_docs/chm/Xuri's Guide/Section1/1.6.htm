1.6 - UOX LINKS
http://uox3.sourceforge.net - The Official UOX3 Homepage.

http://www.uox-3.com - Hosted by DarkStorm, this site contains daily CVS snapshots as well as the latest CVS updates.

http://www.uoxdev.de - The German UOXDev fansite.

http://www.uoxperience.com - A new fansite designed to replace UOXDev.com.

http://www.paigelore.com - Player Run Shard resources, but also a good source for UOX3 stuff (Paige is a good friend of many of the older UOX3 community members).

http://www.beosil.com - Home of Ignition (connection tool) and its packet sniffing plug-in, Sniffy.

http://www.uogateway.com - Home of the connection tool UOGateway.

http://www.smithysanvil.com - A general emulator news site.

http://www.weazy.net - A general UO news site. Also the home of Weazy.

http://dkbush.cablenet-va.com/alazane/insideuo.html - Home to InsideUO, a program that allows you to view data contained within .mul files.

http://www.weazy.net/Accomplice/index.shtml - Accomplice, the GM tool of choice.

http://www.geocities.com/ryandor - soon to be www.ryandor.com. The site has a map making tutorial available along with a Dragon script modification (additional colors for map making)

http://www.uocm.de/index2.htm - For Dragon map making program (German and English site)

http://www.eikon.tum.de/~isp/sphere/doc_index.html - Sphere site but has lots of tools for map making, which applies to all servers.

http://homefree.sensewave.com/~s107293/wb/ - Xuri's WorldBuilder Site

http://homefree.sensewave.com/~s107293/uoxguide/ - Xuri's UOX3 Guide

http://homefree.sensewave.com/~s107293/wb/wforgetutorial.html - Xuri's WorldForge Tutorial
