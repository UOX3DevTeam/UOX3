2.33 - CONTAINER SPAWNER
(Spawning Containers 101 By Iain)

  1. Create a container
  2. Set Type 65 for unlocked, 64 for locked
  3. Set morex equal to the item to spawn
  4. Set morey equal to minimum time to spawn
  5. Set morez equal to maximum time to spawn
  6. Set amount for amount to spawn
