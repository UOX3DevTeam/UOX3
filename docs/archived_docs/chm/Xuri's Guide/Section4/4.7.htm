4.7 - LAYER LIST
Here's a list of the Layers that items can be assigned.

Layer 0x01 - Single-Hand item/weapon
Layer 0x02 - Two-Hand item/weapon (including Shield)
Layer 0x03 - Foot Covering/Armor
Layer 0x04 - Leg Covering (including Pants, Shorts, Bone/Chain/Ring legs)
Layer 0x05 - Chest Clothing/Female Chest Armor
Layer 0x06 - Head Covering/Armor
Layer 0x07 - Hand Covering/Armor
Layer 0x08 - Ring
Layer 0x09 - N/A (not used)
Layer 0x0A - Neck Covering/Armor
Layer 0x0B - Hair
Layer 0x0C - Waist (Half-Apron)
Layer 0x0D - Torso (inner)(Chest Armor)
Layer 0x0E - Bracelet
Layer 0x0F - N/A (no info)
Layer 0x10 - Facial Hair
Layer 0x11 - Torso (Middle)(Surcoat, Tunic, Full Apron, Sash)
Layer 0x12 - Earrings
Layer 0x13 - Arm Covering/Armor
Layer 0x14 - Back (Cloak)
Layer 0x15 - BackPack
Layer 0x16 - Torso (outer)(Robe)
Layer 0x17 - Legs (outer)(Skirt/Kilt)
Layer 0x18 - Legs (inner)(Leg Armor)
Layer 0x19 - Mount (Horse, Ostard, etc.)
Layer 0x1A - NPC Buy Restock Container
Layer 0x1B - NPC Buy No Restock Container
Layer 0x1C - NPC Sell Container
Layer 0x1D - Bank Box
