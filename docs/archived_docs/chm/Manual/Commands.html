The UOX3 Command List
as of, Thursday, Marth 16, 2000

A
ACCT Account management gump
ACTION (h) Perform an animated action sequence.
ADD (hh hh) Adds a new item, or opens the GM menu if no hex codes are specified.
ADDACCOUNT Adds an account **NOT OPERATIVE**
ADDITEM (d) Adds the specified item from ITEMS.SCP
ADDNPC (d or h h) Add the specified NPC from NPC.SCP or by hex ID code
ADDU Opens GM add menu
ADDX (hh hh) Adds a new item at your feet
ALLMOVEOFF Disables GM ability to pick up all objects.
ALLMOVEON Enables GM ability to pick up all objects.
ANNOUNCEOFF Disable announcement of world saves.
ANNOUNCEON Enable announcement of world saves.
 
B
BAN Bans a player
BANACCOUNT Bans an account, place holder
BLT2 Debugging command.
BOLT Select character to bolt.
BRIGHTLIGHT (h) Sets default daylight level.
BUY Select the shopkeeper you'd like to buy from.
 
C
CACHESTATS Display some information about the cache.
CCLEAR Remove the current call from the counselor queue.
CHARC **NOT ACTIVE**
CLEANUP Cleans up corpses and removes excess gates
CLEAR Removes current GM call from queue
CNEXT Attend to the next call in the counselor queue.
COMMAND Executes a trigger scripting command
CQ Display the counselor queue.
CSTATS Provides detailed numerical information about a character
CTRIG (d) Select NPC to trigger.
 
D
DARKLIGHT (h) Sets default night time light level.
DECAY Enables decay on an object.
DISCONNECT (d) Disconnects the user logged in under the specified slot.
DRY Set weather to dry (no rain or snow).
DUNGEONLIGHT (h) Sets default dungeon light level.
DUPE (d / nothing) Duplicates an item. If a parameter is specified, it's how many copies to make.
DYE (hh hh/nothing) Dyes an item a specific colour, or brings up a dyeing menu if no colour is specified.
 
F
FIX Try to compensate for messed up Z coordinates.
FREEZE Select player to freeze in place.
FULLSTATS Select creature to restore full stats.
 
G
GATE Select rune from which to gate.
GCOLLECT Runs garbage collection routines.
GETLIGHT Returns current light level in decimal
GLOW Makes an item glow
GMMENU (d) Opens the specified GM Menu.
GMOPEN (h / nothing) Opens specified layer on player/NPC, or player's pack if no layer specified.
GMS Lists all the GMs online
GMTRANSFER Transfers call to GM Queue
GO (d d d) Go to the specified X/Y/Z coordinates
GOCHAR (h h h h) Teleports you to another character.
GOPLACE (d) Teleports you to a location from the LOCATIONS.SCP file.
GOTOCUR Go to current GM page
GPOST Global post for messageboard
GUARDSOFF Deactivates town guards.
GUARDSON Activates town guards.
GUMPMENU (d) Opens the specified GUMP menu.
GUMPOPEN (h h) Opens the specified GUMP menu.
GY (text) GM Yell - Announce a message to all GMs.
 
H
HEAL Select person to heal.
HIDE Perma-hides someone
HIDEHS Hides the house icons, and displays the house
HOUSE Select location for house. **Won't Work** (links to Targ::ExpPotionTarget())
 
I
INCX Increments the X value
INCY Increments the Y value
INVUL Makes the targeted character immortal.
ISTATS Returns detailed numerical information about an item
ITEMC **NOT OPERATIVE**
ITEMMENU (d) Opens the specified Item Menu.
ITRIG (d) Select item to trigger.
IWIPE (d d d d / nothing) Deletes ALL NPC's and items NOT inside a specified square.
 
J
JAIL Jail's a player
 
K
KICK Kicks a player
KILL Kills a player
KILLALL (d text) Kills %d percent of all characters in the world, and broadcasting text
KILLBEARD Removes the beard from a player/npc
KILLHAIR Removes the hair from a player/npc
KILLPACK Removes the backpack of a player/npc
 
L
LIGHT (h) Sets the light level. 0=brightest, 15=darkest
LOADDEFAULTS Loads the server defaults.
LPOST Local post in messageboard
 
M
MAKECNS Turns a character into a counselor
MAKEGM Turns a character into a GM
MAKESHOP Sets up the 3 layers required for a shopkeeper
MANA Restores mana to the targeted person
MARK Marks a rune
MIDI (d d) Plays the specified MIDI file.
MINECHECK (d) Set the server mine check interval.
MOVETOBAG Select an item to move into your bag.
MUTE (d / nothing) Squelchs specified player.
 
N
NEWX (d) Changes item's x value to d
NEWY (d) Changes item's y value to d
NEWZ (d) Changes item's z value to d
NEXT Moves to next call in the GM queue
NODECAY Prevents an object from ever decaying.
NOINVUL Makes the targeted character mortal.
NPCACTION (h) Select npc to make act.
NPCCIRCLE (d d d) Set bounding circle for a NPC with a NPCWANDER of 4.
NPCRECT (d d d d) Set bounding box for a NPC with a NPCWANDER of 3.
NPCS **NOT OPERATIVE**
NPCSD **NOT OPERATIVE**
NPCTARGET Select player for the NPC to follow.
NPCWANDER (d) Sets the type of wandering a NPC does.
 
P
PCS **NOT OPERATIVE**
PDUMP Display some performance information.
POINT Teleports you to current coordinates, acts like Resend
POLY (h h) Polymorph yourself into any other creature.
POSSESS Select creature to possess.
POST Related to posting in messageboard
 
Q
Q Shows the GM or Counsellor queue.
 
R
RAIN Sets the weather condition to rain.
READINI Re-loads the UOX3.INI file.
READSPAWNREGIONS Re-read the SPAWN.SCP file.
RECALL Select rune from which to recall.
REGSPAWN (d d) Performs a region spawn.
REGSPAWNALL Respawns all region spawns
REGSPAWNMAX (d) Spawns in all regions up to the specified maximum number of NPCs/Items.
RELEASE Releases a player from Jail
RELOADACCOUNTS Reloads the accounts.adm file
RELOADSERVER Reloads the SERVER.SCP file.
REMOVE Removes an item
REMOVEACCOUNT Removes an account, place holder
RENAME (text) Renames any dynamic item in the game.
RENAME2 (text) Sets the item's fully IDed name to text
RESEND Resends server data to client.
RESPAWN Forces a respawn.
RESTOCK Forces a manual vendor restock.
RESTOCKALL Forces a manual vendor restock to maximum values.
RESURRECT Resurrects a character
RPOST Region post with messageboard
 
S
SAVE Saves the current world data into ITEMS.WSC and CHARS.WSC.
SBOPEN Select spellbook to open as a container.
SECONDSPERUOMINUTE (d) Sets the number of real-world seconds that pass for each UO minute.
SELL Select the NPC to sell to.
SEND Debugging command
SET (text, d) Set Str/DEX/INT/Skills on yourself arguments are skill & amount.
SETACCOUNT Sets account, placeholder only
SETADVOBJ Select creature to set ADVOBJ.
SETAMOUNT (d) Select item to edit amount.
SETAMOUNT2 (d) Select item to edit amount2.
SETDIR (d) Select item to edit direction.
SETFONT (h) Select character to change font.
SETHEXMOREXYZ (h h h) Select object to set hex morex, morey, and morez on.
SETID (h h) Select item to polymorph.
SETMORE (h h h h) Select item to edit 'more' value.
SETMOREX (d) Select object to set morex on.
SETMOREXYZ (d d d) Select object to set morex, morey, and morez on.
SETMOREY (d) Select object to set morey on.
SETMOREZ (d) Select object to set morez on.
SETMOVABLE (d) Select item to edit mobiity.
SETMURDER (d) Sets targeted player to have murder count d
SETNPCAI (h) Select npc to set AI type on.
SETOWNER (h h h h) Select NPC or OBJECT to edit owner.
SETPOISON Select creature to set POISON.
SETPOISONED Select creature to set POISONED.
SETPRIV (Chars/NPCs: h h, Objects: h) Sets privileges on a Character or object.
SETRACE (d) Sets targeted player to race d
SETRESTOCK (d) Select item to edit amount to restock.
SETSHOPRESTOCKRATE (d) Sets the universe's shop restock rate.
SETSPADELAY Select creature to set SPADELAY on.
SETSPATTACK Select creature to set SPATTACK on.
SETSPEECH (d) Select NPC to edit speech.
SETTIME (d, d) Sets the current UO time in hours and minutes.
SETTYPE (d) Select item to edit type.
SETVALUE (d) Select item to edit value.
SETVISIBLE (d) Select item to edit visibility.
SETWIPE (d) Select item to modify.
SFX (h h) Plays the specified sound effect.
SHOWDETAIL Shows textual detail about an item
SHOWHS Makes houses appear as deeds.
SHOWIDS Display the serial number of every item on your screen.
SHOWP Displays hex values of your PRIV3 settings.
SHOWSKILLS (d)Shows the skills displayed from skill d for targeted player
SHOWTIME Displays the current UO time.
SHUTDOWN (d) Shuts down the server. Argument is how many minutes until shutdown.
SKIN (h h) Change the hue of your skin.
SNOW Sets the weather condition to snow.
SPAWNKILL (d) Kills spawns from the specified spawn region.
SPLIT Select creature to make able to split.
SPLITCHANCE Select creature to set it's chance of splitting.
SQUELCH (d / nothing) Squelchs specified player.
STAMINA Select person to refresh.
 
T
TELE Select teleport target.
TELESTUFF Select player/object to teleport.
TELL (d) Sends an anonymous message to the user logged in under the specified slot.
TEMP Debugging command.
TILE (h h) Tiles the item specified over a square area.
TILEDATA Dumps tile info about an item to the console
TILEW (h h d d d d d) id id x1 x2 y1 y2 z - Tile an object using specified id & coordinates.
TIME Shows in game time
TITLE (text) Changes the title of any player or NPC.
TRAINER Makes selected NPC a trainer
TTRIG (d) Select item to set trigger type.
TWEAK Select item or character to tweak.
 
U
UNBANACCOUNT Unbans an account, **NOT ACTIVE**
UNFREEZE Select player to unfreeze.
UNGLOW Unglows targeted item
UNHIDE Unhides a creature
USE What object will you use?
 
W
WANIM (h h)Sets lightning animation to parameters
WEB Debugging command.
WF Make the specified item worldforge compatible. ***DOESN'T WORK***
WHERE Prints your current coordinates.
WHO Text version of /wholist
WHOLIST Brings up an interactive listing of online users.
WIPE (d d d d / nothing) Deletes ALL NPC's and items inside a specified square.
WIPENPCS Removes all npcs except for pets and player vendors
WTRIG (word) Sets the trigger word on an NPC.
WSTATS Walking stats of a tile
 
X
XBANK Select target to open bank of.
XGO (d d d) Select char to teleport.
XGOPLACE (d) Send another character to a location in your LOCATIONS.SCP file.
XSBANK Opens special bank of player
XTELE (d / h h h h / nothing) Teleport a player to your position.
 
Z
ZEROKILLS Sets all PK counters to 0.
