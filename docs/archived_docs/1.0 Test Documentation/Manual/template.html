Commands

The commands listed below can be used to access UOX3 functions within the Ultima Online client. Users can access different commands depending on their command level. Users can access all commands with a command level equal to or below their current command level. Up to 50 command levels can be set, providing the greatest amount of security for your shard. To set the command levels, simply find the corresponding line in commands.scp and change it to the desired level. 0 is treated as the lowest level, and 50 is the highest.

Action:
Default Minimum Command Level: 2
Syntax: 'action x (where x is an action number)
The action command makes your character perform action x for the current character doll and then return to the idle animation.

Add:
Default Minimum Command Level: 2
Syntax: 'add xx xx (where xx xx is an item's hex id)
Adds an item with id xx xx at a location of the users choice.
