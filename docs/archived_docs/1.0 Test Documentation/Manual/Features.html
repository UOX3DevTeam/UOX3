Features

UOX has many features, both from OSI shards and from peoples ideas. New features are constantly being added, and old ones are being revised to work more efficiently all the time. The main features of the current UOX release are outlined briefly below.

Definition Format Scripts:

The scripts have been revised and added to in order to come up with the new definition format. It provides more opportunity for customization when working in conjunction with the new JavaScript engine.

JavaScript Engine:

Developed in conjunction with the new scripts, the JavaScript Engine that powers the UOX world is definitely something to be reckoned with. Boasting a huge array of events and functions, and backed by a dedicated script team, its sure to make UOX an enjoyably interactive experience for you and your players.

Stability:

Many sections of UOX's base code have been painstakingly rewritten from scratch, and its paid off. UOX can now handle a far larger player base and do it with great stability.

New Customizable User Interface:

Along with the JavaScript engine and definitions has come a much more navigable in-game user interface. This aids in quick and efficient shard building, or managing in-game events and players problems. These menus can easily be altered though the revised gump system and scripts, which also allows simple creation of new gumps for any custom aspects of your shard.
