Why does my client stop at "Verifying" or "Connecting" ?

There might be several reasons for this. Here are some of the things that may cause these problems:

When halting at Verifying Account:

  • You have put the UOX port (2593) behind your IP address instead of on the line below in UOX3.INI. Remove the Port address from UOX3.INI
  • You are using the wrong UO Client, or the wrong UOX version. Download the latest versions of both programs.
  • You are not using Ignition or UOGateway to start the UO client.
  • The client has crashed.
  • The server may be restarting or booting up.

When halting at "Connecting":

  • You haven't put the UOX port (2593) behind your IP address in LOGIN.CFG. Add the port to LOGIN.CFG, so it says "LoginServer=ip.address,2593" (where ip.address is your IP address )
  • You have put the wrong IP in either LOGIN.CFG or UOX3.INI. Make sure you have the same IP address in both of these files, and also make sure the IP address is valid. Most computers use 127.0.0.1 as the local IP address, or "loopback IP". Use this if you want to play offline, locally on your the same computer as UOX is running on.
    Else, use WINIPCFG (open run, then type in "winipcfg" and press enter) to find your LAN/Internet IP if you intend to use either of those.
  • The Client or the UOX3 server is behind a Firewall which is not set up to let the connections through.
  • The client has crashed.
  • The server may be restarting or booting up.
